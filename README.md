<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
* mariadb


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hstrejoluna/Node-Project-Uptask.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
